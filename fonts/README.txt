This folder contains font files for the Hexadecimal game.

For the game to load these fonts correctly, please ensure you've placed the following font files in this folder:
- AlienCybernetics.ttf (for instructions)
- NeonLights.ttf
- ZappedSticks.ttf
- SurrealismRose.ttf
- NeonFuture.ttf
- DriveThru.ttf
- MadSense.ttf
- NeonAndroid.ttf
- Mama.ttf
- Neonland.ttf
- Monage.ttf
